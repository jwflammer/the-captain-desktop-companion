# the-captain-desktop-companion
Captain is a persistent synthetic companion prototype built first as a desktop computer organism, not a chatbot puppet. He lives as a visible bottom-edge screen companion with motion, internal state, memory, and attention, and is being designed so his identity and behavior can later migrate into other bodies such as an RC crawler or sensor-based robotic platform. The desktop version is the first home body: full screen width habitat, bottom-right resting position, visible movement across the desktop, and shared control between user and system.
